# YouTubeStreamAutoPlay

`open ip:port/?<code>` and this will open a full screen embed of a youtube live stream. 
The codes open a youtube link with the designated youtube ID. If it is a `url`  it will open a link instead. 

# codes
```
{
    "SBL": { "code": "mhJRzQsLZGg" },
    "SCL": { "code": "Jm8wRjD3xVA" },
    "MGL": { "code": "cOmmvhDQ2HM" },
    "surface": { "code": ["inCQ49QGmqQ", "tS2PHJmvJzo", "qw3uaLRrYNY", "TyXUaqscyiU"] },
    "surface-r3ded": { "code": ["inCQ49QGmqQ", "tS2PHJmvJzo", "qw3uaLRrYNY", "IQV0DlS1Ew0"] },
    "iss1": { "code": "fO9e9jnhYK8" },
    "iss2": { "code": "vytmBNhc9ig" },
    "sent": { "code": "IQV0DlS1Ew0" },
    "f12": { "url": "http://192.168.0.194:8000/f12" }
}
```
# Time keeping
- Every 6 hours the steam hard refreshes. 
- Every time the player state is not live it wil refresh the stream. 
# Use case
This is used to keep live streams of Starbase Texas and Cape Canaveral live and up to date with 0 input for years. So far at least one of the devices have been live since Sept 2023.
